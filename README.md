# Plan-de-Dise-o-e-Implementaci-n-de-Infraestructura-de-Red-Empresarial
El Producto 1 consiste en el diseño y planificación de la infraestructura de red para una empresa mediana utilizando tecnologías de conmutación y enrutamiento. El objetivo es desarrollar una red confiable, segura y escalable que permita la comunicación eficiente entre los diferentes departamentos de la organización
